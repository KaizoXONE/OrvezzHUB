# Changelog

All notable changes to OrvezHUB will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-07

### Added

- Initial release of OrvezX UI Library
- Glassmorphism design with black-orange theme
- Auto-save/load configuration system
- Notification system with animations
- Tab system for organizing features
- UI Components:
  - Toggle
  - Button
  - Slider
  - Dropdown
  - Input (TextBox)
  - Label
  - Section
- Drag & resize functionality
- Mobile support
- F3 keyboard shortcut to toggle UI
- Icon library with 30+ icons
- Custom theme support (background images)

### Features

- **Main Window**: Semi-transparent (70% opaque) with blur effect
- **Notifications**: Glassmorphism design with orange accents
- **Responsive**: Works on both PC and Mobile devices
- **Config System**: Version-based auto-save/load
- **Customizable**: Theme colors, tab width, window size

### Fish It Script

- Full automation features
- Integration with OrvezX UI
- Custom configurations
